# Bed & Breakfast Tulipan – Website

Live site: https://thomasbirchler.github.io/bnb_website/

A small static website built to present my mum’s Bed & Breakfast “Tulipan” in Einsiedeln.
Lightweight, fast, and easy to maintain — no framework or backend required.

# Features
- Static HTML/CSS/JS
- Reusable header & footer
- Image slideshows and lightbox
- Google Calendar availability embed
- Individual room pages with clear CTAs

# Tech Stack
- HTML5, CSS3 (responsive grid, mobile-first)
- Vanilla JavaScript (slideshow, lightbox, dynamic includes)

# Run Locally
Clone and open:
```bash
git clone https://github.com/thomasbirchler/bnb_website.git
open bnb_website/index.html
```
